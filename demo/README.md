# Cyto Orbit demo

运行本地演示：

```bash
npm run demo
```

构建静态版本：

```bash
npm run demo:build
```

演示数据来自 [EasonWangs/wordNet-navigator](https://github.com/EasonWangs/wordNet-navigator) 的 [`data/demo-data.json`](https://github.com/EasonWangs/wordNet-navigator/blob/main/data/demo-data.json)，按其 MIT License 使用。原始许可证保存在 `LICENSE.wordNet-navigator`。
